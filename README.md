🎲 Dice Rolling Simulator

This project is a basic dice rolling simulator built to mimic the experience of rolling physical dice. When executed, it generates a random number between 1 and 6, just like a real die.

✨ Features

Simulates rolling a single six-sided die ,
Generates random outcomes using built-in functions ,
Simple and beginner-friendly code structure ,
Can be extended to roll multiple dice or customize dice sides .

🚀 Purpose

This project is designed as a fun way to practice programming fundamentals:
Random number generation.

🛠️ Usage

Run the program, and it will display the result of the dice roll. Perfect for learning, experimenting, or even using in small games
